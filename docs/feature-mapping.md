# Feature Mapping

See [01_FEATURE_MAPPING.md](../01_FEATURE_MAPPING.md).

The important current distinction:

- Whiteboard annotation basics are inherited from upstream and mostly present.
- Realtime collaboration needs Agora configuration and testing.
- Document conversion/upload needs converter, LibreOffice/unoconv, and Cloudinary configuration.
- Audit/reporting has an MVP API and UI, but needs Neon and end-to-end QA before production claims.
